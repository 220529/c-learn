# 🐍 贪吃蛇游戏

SDL2 图形窗口版 - 真正的图形界面，无需终端

## 🚀 运行游戏

```bash
make run
```

或使用脚本：

```bash
./c snake
```

## 📦 打包分发

```bash
make dmg
```

生成 `dist/SnakeGame.dmg` - 可分享给其他 Mac 用户

## 🎮 游戏操作

| 按键          | 功能     |
| ------------- | -------- |
| 方向键 / WASD | 移动     |
| 空格          | 暂停     |
| R             | 重新开始 |
| Q / ESC       | 退出     |

## ✨ 特点

- ✅ 真正的图形窗口（不打开终端）
- ✅ 彩色界面（绿蛇红果）
- ✅ 流畅动画
- ✅ 可打包分发

## 📋 依赖

需要安装 SDL2：

```bash
brew install sdl2
```
