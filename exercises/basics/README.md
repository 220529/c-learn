# 基础语法练习

这个目录包含C语言基础语法的练习。

## 练习列表

1. `hello.c` - Hello World程序，理解基本结构
2. `variables.c` - 变量和数据类型练习
3. `input_output.c` - 输入输出操作
4. `operators.c` - 运算符练习
5. `conditions.c` - 条件语句练习
6. `loops.c` - 循环语句练习

## 编译和运行

```bash
# 从项目根目录编译
gcc -Wall -Wextra -std=c99 -g exercises/basics/hello.c -o hello.exe

# 运行
./hello.exe
```
